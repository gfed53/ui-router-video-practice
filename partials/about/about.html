This is the first incarnation of the Ultimate YouTube Searcher. The goal of this app is to allow the user to gain more freedom in searching for YouTube videos. It is designed to be more lightweight than vanilla YouTube, just as easy to use, but also more powerful when it comes to specifying search parameters

Goals for the (hopefully) not-too-distant future will be a more advanced search feature (obviously, and even more advanced than the vanilla search!), implementation of a playlist feature where users can create a playlist and save it for later, and the ability for users to adjust the size of the embedded video.

This is a modest, one-person project, but I hope that all who use it will get the best experience possible. Enjoy!
